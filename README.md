# AI Trip Planner (Agentic AI)

This project is an AI-powered trip planning system built using Python, CrewAI, and OpenAI APIs.  
It uses multiple autonomous agents to collaboratively generate personalized travel recommendations.

## Features
- City selection based on weather, season, and budget
- Local expert recommendations
- Travel planning with budget and packing suggestions
- Multi-agent collaboration using CrewAI

## Tech Stack
- Python
- CrewAI
- OpenAI API
- LangChain
